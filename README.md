# Model Serving Platform with TensorFlow Serving & TorchServe

## Overview

This project exposes machine learning models using **TensorFlow Serving** and **TorchServe**
running on a **Kubernetes** cluster.  
The goal is to provide a unified model serving platform with separate services for
TensorFlow and PyT
